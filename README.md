# dictcord
## Configuration
1. Create a Discord Bot
2. Create a Test Server and add your bot in it
3. Update config.json with the token and client id of your discord bot
4. Update config.json with the guild id of your test server
5. Execute command `node deploy.js` to deploy the commands to your test server
6. Execute command `node index.js` to run the bot
7. In your Test Server run `/dictcord` to get the prompt to enter the word for which you need the meaning
8. See [Discord Dictionary Bot](https://www.anshisoftlabs.com/portfolio/dictcord/) for details and Demo
